# Django To Do List Application #

It allows users to 
* Create Tasks<br>
* Update Tasks<br>
* Mark Tasks as Completed<br>
* Delete Tasks<br>

## Technologies Used : ##

* Front-End :  HTML, CSS<br>
* Back-End : Python Django<br>
